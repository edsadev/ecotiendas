# Modulos

El proyecto Ecotienda consta de 4 grandes modulos los cuales se detallan acontinuacion:

1. **Backend/API**: El cual se encuentran todas las funciones o endpoints para que el frontend pueda interacturar con la base de datos esto se lo conoce como API.

2. **Frontend/Webapp**: Se encarga de presentar la interfaz web para los usuario Ecoadmin, keyUser, Administrativo, EcoZonal.

3. **EcoAmigo/App**: Es la aplicacion que utilizaran los usuarios.

4. **EcoPicker/App**: Es una aplicacion coorporativa la cual la usaran los recolectores de material reciclable.
